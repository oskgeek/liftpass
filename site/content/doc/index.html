What is Liftpass?

Liftpass makes managing prices of virtual goods and rewards for IAP easier. Get started by installing the server, and jump into our getting started guide to start managing your prices and rewards.

  1. Get Started How to install the Liftpass server on any Linux based system in minutes.
  2. Content Management Understand the aspects of an application that Liftpass manages.
  3. SDK Implementation Start sending new prices to users with the Liftpass SDK.
  4. API Reference Manage the application content and prices with the full featured API.

How Liftpass works

Fundamentally Liftpass manages prices of virtual goods and the rewards for in-app purchases. In conjunction with the SDK, Liftpass enables developers to iterate on those prices and target specific users with customized prices based on their progress in the application.

At the moment the SDK requests a new set of prices the Liftpass server figures out based on the analytics recorded about the user which set of prices the user should receive.
